What is Hardware? Hardware refers to the physical components of a computer network, such as devices and equipment used to build and operate the network.

Glossary

Node
Devices that create, send, receive, or forward data on a network, including computers, phones, sensors, routers, printers, and microcontrollers.
Switches
Smarter devices that send data only to the intended device using MAC addresses, reducing collisions and improving network performance.
Hub /Smart Hub
Basic devices that broadcast data to all connected computers, causing many collisions. Rarely used today due to inefficiency. / Central controllers for smart home devices, using Wi‑Fi, Bluetooth, Zigbee, or Z‑Wave to coordinate automation and communication.
Router
Devices that direct data between networks, choosing the best path for packets. Connect homes to ISPs and form the backbone of the internet.
Wireless Access Points (WAPs)
Allow wireless devices to join a wired network, enabling laptops, phones, and tablets to access LANs and the internet.
Servers and types of servers
Powerful computers provide services like file storage, printing, websites, email, and databases to clients on a network. Types include web servers, file servers, mail servers, etc.
